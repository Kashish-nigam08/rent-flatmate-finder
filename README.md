# AI-Powered Rent Flatmate Finder

## Features

- JWT Authentication
- Owner & Tenant Roles
- Room Listings
- Tenant Profile
- AI Compatibility Matching
- Interest Requests
- Real-Time Chat
- Email Notifications

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS

Backend:
- FastAPI
- SQLAlchemy
- MySQL

Authentication:
- JWT

AI:
- Google Gemini API

## Installation

### Backend

cd backend

pip install -r requirements.txt

uvicorn app.main:app --reload

### Frontend

cd frontend

npm install

npm run dev
